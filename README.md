# Compressor Health Monitoring Dashboard

An interactive dashboard for monitoring industrial compressor health using Python and Streamlit.

## Features

- Compressor-wise monitoring (C1, C2, C3)
- Health score calculation
- Maintenance alerts
- Recommended actions
- Pressure trend visualization
- Temperature trend visualization
- Vibration trend visualization

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Objective

To demonstrate condition monitoring and preventive maintenance concepts for industrial compressors using data analytics and interactive visualization.
